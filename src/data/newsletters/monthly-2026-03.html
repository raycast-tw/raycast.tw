三月更新：美國活動、Windows 與 iOS 更新、Glaze Sessions

這期月報由 Pedro 帶來，整理了三月最重要的產品更新與社群消息。除了 Windows 與 iOS 的功能進展，Glaze 上線一個月後的使用成果與美國活動計畫，也都是本期重點。

Raycast for Windows

Windows 團隊這個月一共推出了五個版本，整體體驗正快速朝向與 Mac 版接近的功能完整度前進。

  • 翻譯指令：可直接在 Raycast 中快速翻譯文字，官方也建議綁定像 Win + T 這樣的快捷鍵來提高使用頻率。
  • Quick Look：在檔案搜尋中按下 Ctrl + Y，即可即時預覽檔案內容，不必另外打開 App。
  • 視窗管理指令：可透過 Create Command 建立自訂視窗大小與位置，讓桌面整理更有效率。
  • 虛擬桌面指令：可將 Open Desktop 1-10 綁到 Alt + 1 到 Alt + 0，也能把 Move to Desktop 1-10 綁到 Alt + Shift + 1 到 Alt + Shift + 0，快速切換或移動視窗。
  • 錯誤修正與優化：更多細節可參考 Windows 更新日誌。

Raycast for iOS

iOS App 迎來了許多人期待已久的 Raycast Calculator，將桌面版熟悉的計算體驗帶到手機上。

  • 自訂圖示：Quicklinks、AI Commands 與 Presets 現在都能選擇自訂圖示。
  • 平行背景聊天：可以同時執行多個背景對話，讓行動端的 AI 使用流程更順手。
  • 錯誤修正與優化：完整變更可參考 iOS 更新日誌。

Glaze Sessions

Glaze 上線滿一個月，官方表示已經有許多團隊與個人完成導入，並且累積打造出數百個 App。Raycast 也同步啟動了 Glaze Sessions 系列活動，讓更多人能直接體驗產品並取得使用機會。

目前已在倫敦、巴黎與柏林舉辦過活動，參與者通常只有一到兩小時的時間可以實作，但現場做出的成果已經讓團隊相當驚艷。

其他消息

  • Raycast 首次前進美國：官方將首次在舊金山與紐約舉辦活動。
  • 舊金山活動：包含 The New Standard with Cursor and Vercel，以及 Raycast x Vercel Builders Night。
  • 紐約活動：包含 The New Standard with Linear and Anthropic，以及 Glaze Sessions。
  • 社群行事曆：所有即將舉辦的 Raycafe 都整理在 Raycast Community Calendar，如果你的城市還沒出現，也可以申請成為 Raycast Ambassador 自行主辦。
  • 內容更新：Pedro 在個人 YouTube 頻道分享了他如何用 Glaze 解決自己的工作流程問題；Pedro 與 Thomas 也透過直播說明新版 Raycast 的 rollout 狀況。
