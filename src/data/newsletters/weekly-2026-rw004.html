本週新上架擴充

  • LobeHub Icons：熱門 AI 與 LLM 品牌的 SVG logo 與 icon 集合。作者 ridemountainpig。另有 Demo。
  • Datafast：查看 Datafast 網站分析資料，包括訪客、收入、頁面、來源、活動與即時數據。作者 joshmillgate。另有 Demo。
  • Kill MCP Servers：查看並管理 Mac 上執行中的 MCP servers，也能檢查 RAM 使用量與終止程序。作者 baptiste-jn。另有 Demo。
  • Tesla Energy：監看 Tesla 太陽能板與 Powerwall，即時查看發電、電池、電網與歷史能源資料。作者 chrismessina。
  • In the Timezone：視覺化比較多個時區，並可拖曳時間檢視差異。作者 i_idz。
  • Voice-to-Text for Windows：錄下麥克風語音，透過 OpenAI Whisper 轉錄，並用 AI 清理文字。作者 rennesis。
  • VocaBuilder：在不同語言之間翻譯單字並建立自己的字彙庫。作者 AngryBeaver。
  • Portfolio Tracker：追蹤股票、ETF、不動產、現金、負債與加密資產價值，自動更新價格並查看淨值變化。作者 filipawaits。
  • Moodist：用於專注與放鬆的環境聲混音器，可疊加多種聲音、儲存 preset，並從選單列控制。作者 chandlervdw。
  • Codex Manager：用安全的本地操作管理 Codex 的 MCP servers 與 skills。作者 jose_de_freitas。
  • Lunaris：查詢 Genshin Impact 的角色、武器與聖遺物資訊。作者 sphygmus。
  • cmux：與 Mac 上的 cmux terminal app 互動。作者 steve_occhipinti。
  • Display Input Switcher：透過 Raycast 與 m1ddc 切換螢幕輸入來源。作者 clins1994。
