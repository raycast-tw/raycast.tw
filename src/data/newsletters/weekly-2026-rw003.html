本週新上架擴充

  • RAM Prices：追蹤來自 RamRadar 的 DDR4 與 DDR5 市場價格。作者 alexi.build。另有 Demo。
  • Notra：直接從 Raycast 檢視、管理與建立 Notra 內容。作者 dominikdev。
  • Trimmy：把你複製的多行 shell 指令攤平成真正可貼上執行的一行版本。作者 chrismessina。
  • Beardtown：查看 Beard Meats Food 這位英國競食者的挑戰統計與資訊。作者 bensomething。
  • Shape Calendar：用 Shape Calendar 規劃訓練生活，可搜尋、建立與管理活動。作者 samuelkraft。
  • Git Batch Tools：對多個 project groups 批次執行 `git pull`、`git status` 等操作。作者 Logan-8f。
  • Thaw：Thaw 的選單列管理工具。作者 diazdesandi。
  • Voicenotes：直接從 Raycast 搜尋與管理 Voicenotes。作者 Parzival。
  • Parachord：控制 Parachord 音樂播放器，可播放、暫停、跳歌、搜尋，還能和 AI DJ 對話。作者 parachord。
  • Watchkey：用 Touch ID 與 Apple Watch 儲存及取得 keychain secrets。作者 etheirystech。
  • Korean Add Calendar：用韓文自然語言建立 Apple Calendar 活動或提醒事項。作者 hjm79。
  • Databuddy：隱私優先的網站分析與短網址工具，可查看網站、追蹤訪客並管理短連結。作者 izadoesdev。
  • Clockodo：在 Raycast 中查看你的 Clockodo 紀錄。作者 leomelzer。
  • Tmux Cheatsheet：在忘記指令或剛學習時，快速查閱 tmux cheatsheet 文件。作者 proprocrastinator。
  • Bundles：把 app、網站與巢狀集合整理成自訂 bundles。作者 Jace。
  • Habitica ToDos：非官方 Habitica extension，可查看並建立 ToDo。作者 Sacrachan。
  • Caaals Food Tracker：用 AI 記錄飲食並瀏覽你的飲食日誌。作者 chuck-38。
  • nocal：用 nocal 在 Raycast 中搜尋筆記並管理日程。作者 nocal。
  • Paper Agent：從本地 Paper Agent workflow 中探索、整理與管理論文。作者 marvel940804836。
  • Kafka UI：檢查 Kafka consumer group lag、瀏覽 topics，並開啟 Kafka UI dashboard。作者 NirRosh。
  • Windows Environment Variables：查看、搜尋、編輯與管理 Windows 環境變數及 PATH 項目，以鍵盤操作取代傳統設定視窗。作者 ohmycodedev。
  • S.EE：建立短網址、分享文字並上傳檔案。作者 missuo。
  • Google Contacts：直接從 Raycast 瀏覽、搜尋與管理 Google Contacts，可建立、編輯、刪除並操作聯絡人。作者 aic。
  • Marble：管理 Marble 的貼文、作者、分類與標籤。作者 dominikdev。
