Raycast Weekly RW002

第二期 Raycast Weekly 聚焦自家 newsletter extension、Grok 4.20、iOS 與 Windows 的功能更新，以及一批新上架的社群 extension。

完整原文可參考 Raycast Weekly. Issue RW002。

上週焦點

  • Raycast Weekly Newsletter 現在有自己的 Raycast extension。
  • Grok 4.20 已可在 Raycast 中使用。
  • Raycast iOS 現在支援 Quicklinks、AI Commands 與 Presets 的自訂圖示。
  • Translator 現在已登上 Raycast for Windows。
  • Dominik 分享了由 Raycast 驅動的 Codex workflow。
  • Alex 用 Glaze 打造 Hotkey Explorer，把 Raycast hotkeys 視覺化。
  • Raycafe extension 幫你追蹤 Raycast 社群活動。
  • Alexi 的 #100DaysOfRaycast 持續推進，這期收錄了 Day 161、Day 162、Day 163、Day 164、Day 165、Day 166 與 Day 167。

Raycast Ambassador Program

如果你想在自己的城市主辦 Raycafe，或單純很愛向別人推薦 Raycast，也可以看看 Raycast Ambassadors 計畫。

本週新上架擴充

  • Raycafe：瀏覽即將到來的 Raycast Community meetup，並直接開啟活動頁面。作者 lemikeone。另有 Demo。
  • Nano Games：在 Raycast 搜尋列中用點字字元玩 Snake 與其他小遊戲。作者 alexi.build。另有 Demo。
  • Apple Photos：以格狀方式瀏覽近期 Apple Photos，並把最新照片直接貼進任何 app。作者 wesbaker。
  • Minecast：快速查看 Minecraft 伺服器狀態。作者 Dori7os。
  • Now Playing：在 Raycast 選單列顯示目前播放的歌曲與藝人。作者 kendaniels。
  • No More Caffeine：估算體內剩餘咖啡因，並判斷再喝一杯是否會影響睡眠。作者 atsushi_kawamura。
  • Auth0 Management：跨多個 tenant 搜尋與管理 Auth0 使用者。作者 ray_tan。
  • Link Transformer：用自訂 JavaScript actions 儲存、搜尋並轉換連結。作者 dangkhoipro。
  • Music Recognition：透過 macOS Shortcuts proxy 辨識音樂，也就是用 Shazam 的方式找歌。作者 lemikeone。
  • Navidrome：搜尋並瀏覽你的 Navidrome 音樂庫，快速開啟藝人、專輯與歌曲。作者 alexandervarney。
  • Phare.io Uptime：管理並查看使用 Phare.io 的服務 uptime 狀態。作者 ApopeOS。
  • Petal - Offline Voice to Text：控制 Petal 的錄音、歷史搜尋、模型切換與最後一筆轉錄複製。作者 Aayush9029。
  • Kommand：從 Kommand library 顯示目前前景 app 的鍵盤快捷鍵。作者 umit_gul。
  • Quarantine Manager：檢查 macOS 檔案與 app 的 extended attributes、查看 quarantine 狀態，並移除已驗證安全檔案的 `com.apple.quarantine`。作者 nurkamol。
  • Leave Time Calculator：根據上班時間與休息長度計算下班時間。作者 harunonsystem。
  • SQL Reference Search：提供 PostgreSQL、MySQL、SQLite 與 T-SQL 的方言感知 SQL 參考，包含語法、範例與實用 query 模式。作者 FariaF22。
  • Pandas Documentation Search：快速搜尋官方 Pandas 文件。作者 FariaF22。
  • Xbox Friends：查看哪些 Xbox Live 朋友在線上，以及他們正在玩什麼。作者 MaxPaolucci。
  • Público：直接在命令列中接收 Público 的最新新聞。作者 caasols。
  • Sadaqah Box：管理你的 charity boxes 與 sadaqahs。作者 emircanerkul。
  • TabStash：搜尋、瀏覽並收藏書籤。作者 mkhnsn。
  • Colima：直接從 Raycast 管理 Colima 虛擬機實例與 Docker 環境。作者 MiskaMyasa。
