上週最值得關注

  • Raycast for Windows v0.54 導入了 Hyperkey，讓單一按鍵可以化身成強力的快捷鍵修飾鍵。
  • 33 種 Hyperkey 用法：社群整理了大量實際範例，幫助你把常用操作映射成更高效率的快捷鍵。
  • 活動預熱：Raycast 將與 Vercel 在舊金山舉辦 Builders Night，也會與 Cursor、Vercel 一起進行公開對談。
  • 社群聲量：MKBHD 被提到也是 Raycast 使用者，同時 Raycast 在 X 上的追蹤數突破 90,000。
  • Glaze 作品：Olli 展示了以 WHOOP 健康資料打造的 App；Yiwei Ho 分享了 Repo Scout 與網站相片管理工作流；Yen Cheng Lin 的 Raycast Maze 也被列為本期亮點之一。
  • #100DaysOfRaycast：Alexi 繼續挑戰，從第 195 天一路更新到第 201 天，並在第 200 天完成第二季。

本週新上架擴充

  • Music News：整合多個來源閱讀最新音樂新聞。另有 Demo。
  • Zread.ai：快速打開 GitHub repository 的 zread.ai 頁面，並瀏覽熱門專案。另有 Demo。
  • Icy Veins Quicklinks：快速前往 Icy Veins 的 World of Warcraft 職業與天賦指南。
  • Cloudstash：把連結快速存進 Cloudstash。
  • Have I Been Pwned：查詢電子郵件或密碼是否出現在外洩資料中。
  • FreshRSS：瀏覽與管理 FreshRSS 文章。
  • OpenRouter Manager：查看 OpenRouter 點數並管理 API Keys。
  • Clean Agent Text：清理從 AI agent 終端介面複製出來的文字，移除框線符號與多餘空白。
  • Voicemeeter Control：直接用 Raycast 控制 Voicemeeter 的 strip 與 bus。
