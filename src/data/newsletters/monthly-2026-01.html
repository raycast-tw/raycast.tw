一月更新：Typing Practice、Chat Branching 與 Team Picks

這是 2026 年的第一封 Raycast 月報。Pedro 提到團隊同時忙著推進新版 Raycast for Mac、Windows 與 iOS，而這期最明顯的更新重心落在 Windows 的新互動功能，以及 iOS 聊天體驗的延伸。

Raycast for Windows

Windows 自公開 beta 之後持續收到大量更新，Pedro 也特別提到這段時間的使用者回饋相當好。

  • Typing Practice：Raycast 與 Monkeytype 合作，把打字練習帶到 Windows。可透過 Start Typing Practice 指令開始，並在 Quote 與 Time 模式中切換。
  • AI Commands：可以用簡單文字提示自動化重複工作，除了內建像 Fix Spelling and Grammar 這類指令，也能自建。官方還提供了 Prompt Library。
  • Windows Run：以更現代的方式取代傳統 Run 對話框，讓你不必硬背指令。
  • Fallback Commands：Root Search 沒有結果時，現在可自訂顯示哪些 fallback 行為。

Raycast for iOS

iOS 這期的關鍵字是更靈活的聊天流程。

  • Chat Branching：你可以從任何一則訊息分支出替代對話路徑。
  • Disposable Chats：可以快速開啟一次性對話，不會把歷史紀錄弄得太雜。
  • 從 ray.so 匯入：現在能從 ray.so 匯入 Quicklinks、Snippets、AI Commands 與 Presets，也能建立可分享的 ray.so 連結。
