二月更新：Introducing Glaze、新版 Raycast 時程與更多更新

這封月報的主軸很明確：Pedro 終於公開了 Glaze，也同步帶來 Windows、iOS、直播與 Glaze Sessions 的最新消息。這也是 Raycast 產品線持續擴張的一期。

Introducing Glaze

Glaze 是 Raycast 對「透過和 AI 對話來建立桌面 App」這件事的答案。官方強調，這些 App 不只是能用，而是會像真正屬於你 Mac 的原生桌面軟體。

Raycast 團隊認為，他們過去六年在桌面軟體上的經驗，都被濃縮進了 Glaze。目標是讓不會寫程式的人，也有機會打造自己的桌面應用。

Raycast for Windows

Windows 在這個月繼續快速追趕 Mac 版功能，官方也明講目標仍然是逐步達成功能對齊。

  • Raycast Notes：現在已在 Windows 上線。Pedro 特別推薦為它綁一個快捷鍵，例如 Win + N。
  • Calculator History：所有計算與轉換紀錄都會保留下來，而且可搜尋、可釘選。
  • Search Screenshots：可搜尋截圖與螢幕錄影，甚至可以直接搜尋圖中的文字。
  • Script Commands：常用腳本可以直接從 Raycast 執行，也能綁定快捷鍵。
  • Networking Indexing：File Search 現在支援索引網路磁碟與可移除裝置。

Raycast for iOS

iOS 團隊這個月也交出很扎實的更新內容，尤其是背景聊天體驗的提升。

  • 多個背景聊天：現在可以同時讓多個對話在背景執行。
  • Disposable Chats：適合處理一次性問題，不會把聊天紀錄越堆越亂。
  • 從 ray.so 匯入：可以直接從 ray.so 匯入 Quicklinks、Snippets、AI Commands 與 Presets。
  • 穩定性與修正：更多細節可參考 iOS 更新日誌。
